WP Permly uses the Permly API to generate short links for all your posts and pages. It can be used to email, share, or bookmark the posts or pages quickly and easily.

== Description ==

WP Permly helps you to generate short links using the Permly API for all of your blogs posts and pages.

== Installation ==

Installation of WP Permly is as easy as:

1. Upload the entire `/wp-permly/` folder to your `/wp-content/plugins/` directory.
1. Navigate to the 'Plugins' page of your dashboard, and activate WP Permly
1. After activation proceed to the WP Permly settings page and configure your Permly API key.
1. A new metabox will appear on the options page to generate short links for all your existing posts and pages.
1. New posts will automatically generate shortlinks as you create and publish them!
